## Installation

Copy main_{algorithm}.cpp to main.cpp , replace it and build.
 
```bash
cp main_dynamic.cpp main.cpp
make
```

## Usage

Execute binary with arguments.

Argument list can be find in main.cpp file.

```bash
./a.out {arguments}
```
